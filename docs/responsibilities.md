# Responsibilities

This document describes the responsibilities undertaken throughout the development of the ADAS Smart Speed Bump Detection & Localization System.

---

# Project Organization

The project was developed by a multidisciplinary engineering team covering artificial intelligence, embedded systems, backend development, mobile development, documentation, and system integration.

My primary role focused on Software Architecture and Technical Documentation.

---

# Primary Role

## Software Architecture Engineer

Responsible for designing the overall software architecture of the system and defining the interaction between all project components.

### Responsibilities

- Designed the complete software architecture.
- Defined communication between AI, Backend, GPS and Mobile App.
- Planned project modules and dependencies.
- Designed project workflows.
- Organized repository structure.
- Prepared architecture documentation.
- Defined API communication strategy.
- Planned scalability for future versions.

---

## Documentation Engineer

Responsible for producing professional engineering documentation for the project.

### Responsibilities

- Wrote project documentation.
- Created technical specifications.
- Designed workflow diagrams.
- Prepared deployment documentation.
- Wrote GitHub repository documentation.
- Prepared engineering reports.
- Organized project references.
- Created hardware and software documentation.
