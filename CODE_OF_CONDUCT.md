# Code of Conduct

We are committed to providing a welcoming environment for everyone.

---

## Expected Behavior

- Be respectful.
- Be professional.
- Accept constructive feedback.
- Help others learn.

---

## Unacceptable Behavior

- Harassment
- Offensive language
- Personal attacks
- Discrimination

---

## Enforcement

Project maintainers may remove inappropriate contributions.

---

Thank you for helping make this project welcoming for everyone.
